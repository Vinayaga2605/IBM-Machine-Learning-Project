# 🚀 Rural Infrastructure Project Classification Using Machine Learning

## 📖 Project Overview
This project was developed as part of the **IBM SkillsBuild AICTE Machine Learning Internship** using **IBM watsonx.ai Studio** and **IBM AutoAI**.

The objective of this project is to classify **Rural Infrastructure Projects** into the appropriate **PMGSY categories** using Machine Learning. IBM AutoAI was used to automate data preprocessing, feature engineering, model selection, and hyperparameter optimization.

---

## 🎯 Problem Statement
Develop a Machine Learning model that accurately classifies rural infrastructure projects based on the **PMGSY dataset**, helping improve project categorization and decision-making.
